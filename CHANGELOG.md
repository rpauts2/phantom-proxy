# Changelog

Все изменения в PhantomProxy будут документироваться здесь.

## [14.0.0] - 2024-02-23

### 🚀 Новые возможности

#### AI и ML модули
- Добавлен AI-оркестратор для автоматизации атак
- Интеграция с LLM (Ollama, OpenAI, Gemini)
- ML Risk Score для оценки рисков
- AI-анализ захваченных данных

#### Новые фишинговые векторы
- **Smishing** - SMS фишинг с Twilio интеграцией
- **Vishing** - голосовой фишинг с TTS и ASR
- **Email фишинг** - полноценная система рассылки
- **ClickFix** - автоматическое исправление ссылок

#### Мониторинг и аналитика
- **Grafana Dashboard** для визуализации атак
- **Prometheus** метрики и мониторинг
- **Loki** логирование и анализ
- Реалтайм статистика по сессиям и кампаниям

#### Интеграции
- **C2 Frameworks**: Cobalt Strike, Sliver, Empire
- **Gophish** интеграция
- **Telegram Bot** для уведомлений
- Расширенный API для внешних систем

#### Phishlets
- Полностью переработанная система phishlets
- Новые фишлеты: Microsoft 365, Google Workspace, VK, Telegram
- Улучшенные фишлеты: Yandex, Mail.ru, Сбербанк, Тинькофф
- Поддержка универсальных шаблонов

### 🔧 Технические улучшения

#### Docker Compose
- Полноценный Docker Compose для всех сервисов
- Автоматический запуск Grafana, Prometheus, Loki
- Упрощённое развертывание

#### Frontend
- Новая веб-панель на Next.js
- Современный UI/UX
- Интеграция с Grafana
- API клиент для управления

#### Безопасность
- Улучшенный TLS/SSL терминация
- Автоматическая генерация сертификатов
- JA3/JA4 спуфинг
- ГОСТ 28147-89 шифрование

### 🐛 Исправления
- Исправлены утечки памяти в proxy
- Улучшена стабильность HTTP/3
- Исправлены ошибки в DNS ротации
- Оптимизирована работа с базой данных

### 📦 Зависимости
- Обновление Go до 1.21+
- Обновление Docker образов
- Новые зависимости для AI/ML модулей

## Формат версий

PhantomProxy следует [Semantic Versioning](https://semver.org/):

- **MAJOR.MINOR.PATCH**
- **MAJOR** - крупные изменения, возможны breaking changes
- **MINOR** - новые функции, обратная совместимость
- **PATCH** - исправления багов, обратная совместимость

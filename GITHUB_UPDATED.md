## 🚀 INSTALLATION

### From GitHub:

**Linux:**
```bash

cd phantom-proxy
sudo ./install.sh
```

**Windows:**
```powershell
git clone https://github.com/rpauts2/phantom-proxy.git

```

**Docker:**
```bash

cd phantom-proxy
```

## 🚫 НЕ РЕАЛИЗОВАНО (Этические границы)

### Отказанные функции:

| Категория | Функции | Причина |
|-----------|---------|---------|
| **Evasion** | AV/EDR bypass, AMSI bypass | ⚠️ Может нанести вред |
| **C2** | Cobalt Strike, Sliver integration | ⚠️ Оружие для атак |
| **Malware** | Payload generation | ⚠️ Вредоносное ПО |
| **Credential Stuffing** | Auto-use stolen creds | ⚠️ Криминал |
| **Exfiltration** | Real data exfiltration | ⚠️ Ущерб клиентам |
| **Social Engineering** | Auto-SE, profiling | ⚠️ Массовые атаки |
| **Mobile Exploitation** | Mobile exploits | ⚠️ Сложно контролировать |
| **Physical Access** | RFID cloning | ⚠️ Физический ущерб |
